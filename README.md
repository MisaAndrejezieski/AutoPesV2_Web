## 📋 Sobre o Projeto

AutoPes V2 é uma ferramenta desktop que automatiza pesquisas no Microsoft Edge, gerando relatórios em CSV com os resultados.

Desenvolvido com **Python**, **PyAutoGUI** e **Tkinter**.

## 🛠️ Como Usar

### 1. Baixe o executável
Clique no botão de download no site oficial

### 2. Execute o programa
Dê dois cliques em `AutoPesV2.exe`

### 3. Configure a automação
- Escolha o número de **TEMAS**
- Escolha o número de **PERGUNTAS POR TEMA**

### 4. Inicie
Clique em **"INICIAR"** e aguarde

### 5. Resultados
Os resultados são salvos automaticamente em CSV na pasta `data/resultados/`
